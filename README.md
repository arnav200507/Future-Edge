Future Edge is a student counselling and admission management portal designed to streamline the admission process for students and counsellors.

## Features
- Student login using registration number
- Admin dashboard for student management
- Admission progress tracking
- Document upload & verification
- Form upload & download
- News & updates management

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Supabase (Auth, Database, Storage)

```bash
npm install
npm run dev
